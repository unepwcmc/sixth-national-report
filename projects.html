Find data

Within the global UN Biodiversity Lab project, you can filter views by keyword by entering your search term(s) in the “Filter Views” box. Views can be sorted according to time or date. You can choose to filter by type of data in components, keywords in classes, or by project theme in collections.

Overlap Analysis

Overlap analysis enables the user to create a view showing the overlap of two other views. The user can then return values about area and feature values occurring within the overlap. This tool is under development and will be released Summer 2018. Stay tuned!

Watch this video by the MapX team to help walk you through uploading your own data and styling your own view.

Download Data

Click the cloud button in the toolbar beneath the view to download the data. Please note that not all datasets are available to download - some are protected as view only licenses. You can customize the data by selecting the country you would like to clip (spatially select) the data to, and the output format of the data.

Export Maps

Click the camera button (“create a screenshot”) in the toolbar beneath the view to export your maps to add to your reports, publications or presentations. When you export a map, a zip file will be created. Within this zip file are 4 elements: three PNG files including the scale, map image and legend, and a PDF which has all of the elements combined. If you resize the map afterwards, you must resize the scale as well to retain accuracy.

Share Results

To share your results, select the “Share Data” button in the toolbar beneath the view. Within the window that appears you can choose to receive a direct link to your selected view, with customizable options such as setting the project, collection, or view of the link. To receive an HTML code for embedding the link into a website or blog using an iframe, click "Include URL in an iframe". You can then adjust the width or height of the iframe (in pixels) within the code.
