Welcome to your sandbox

Welcome to your Sandbox! Here you will have access to additional data layers, analytical tools, and the ability to upload data, download data, export and share your maps. Sandboxes can be created for each project you are working on. You can also invite or be invited to the sandboxes of colleagues for collaboration and co-design.

Start

Find data

Searching and filtering data are essential to help find what you are looking for. You can filter data by theme here. You can also filter by keyword or type of data (vector, raster, story map, dashboard) if you prefer.

Example:

Overlap Analysis

This is where you can perform an overlap of selected features. When you click the button (area of visible features) the number it will return is the area of all the features you have selected in your view. If you want to perform an overlap analysis, this will return a figure with the total area of overlap between layers.

Example:

Upload Data

In order to upload your own data, it must be compatible with an open source format. This means it must be an ESRI Shapefile, GeoJSON, GPX, or KML file. A GeoJSON format is recommended. Go to the relevant country or world view and simply drag and drop your file into the frame. If this doesn’t work, you can go to the toolbox, select “add a source” and manually upload the data once the upload is complete you’ll need to filll out the metadata section.

Example:

Download Data

To download datasets you can select which format you would prefer to have it downloaded in, if it is available in that format.

Example:

Export Maps

This is where you can export your maps to add to your reports, publications, presentations, or whatever you choose. When you export a map, a zip file will be created. Within this zip file are 4 elements: three PNG files with the scale, map image and legend, and a PDF which has all of the elements combined. Don’t forget, if you resize the map you must resize the scale as well!

Example:

Share Results

When you want to share your results online, you have a few options. You can customize an HTML code, use an iframe, or share the unique identifier of each map. These buttons accomplish each of these goals.
